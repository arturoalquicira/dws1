Compare & Contrast Zen Garden Designs

I have chosen three different designs of some Web Designers from the website Zen Garden to compare and contrast the difference between their CSS styles.

The designs that I chose were: Make'em Proud, CSS Co,. LTD, and Under The Sea

Fonts & Headings

I see that the three designs set their font styles and sizes at the beginning on the body part trying to making it more general, but the design Make'em Proud sets other elements too besides the body like (div, span, applet, object, iframe, h1, h2, h3, h4, h5, h6, p, blockquote, pre, etc)

But the similarities are that the three designs use the id or classes to specify where they want to change font style or size

Text Colors & Background Colors

I have noticed that these design don't change often their text colors, they are constant, but sometimes they change it to remark something and do not use the strong element. For example CSS Co,. LTD design changes the color of the abbreviations using classes and the element abbr like this: .preamble abbr{.

For the background colors they specify where do they want to change the color using the selector background and the declaration will be the color they prefer.

Since they are several frames or divisions where information is displayed, they specify the background colors using the div class.

Background Images

The background images seems to be more easy to insert, as long as you specify where you want to use them, normally you use the selector background and the declaration will be the url or the path where the image is located.

Layout

The layouts are different in each design and the layout normally refers to the padding, margin, align, float and other selectors that help you give like a kind of format to any text or image that you want to display on the page.
