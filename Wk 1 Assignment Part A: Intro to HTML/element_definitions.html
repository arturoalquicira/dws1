Element Definitions

This is a short summary to define each of the elements introduced in the textbook HTML & CSS: Design and Build Websites.

Chapter 2

Headings
HTML has six levels of of headings:
<h1> is used for main headings.
<h2> is used for subheadings.
Paragraphs
To create a paragraph, start the words that make up the paragraph with an opening <p> tag and closing </p> tag."
Bold & Italic
To turn a character or a word in bold, enclose it in the tags <b> and </b>.
To turn a character or a word in italics, enclose it in the tags <i> and </i>.
Superscript & Subscript
Characters that should be superscript as suffixes such as dates or mathematical concepts use the element <sup> and </sup>.
Characters that should be subscript as foot notes such as chemical formulas use the element <sub> and </sub>.
White Space
Also known as white space collapsing, some developers add more than one spaces in order to insert more elements, but it won't change anything in the navigator. It will only display one space.
Line Breaks & horizontal Rules
If you want to add a line break into the middle of a paragraph you can use at the end of it the tag <br />.
Visual Editors & The Code Views
Visual editors are word proccesors that allow you to control the presentation of the text.
Code Viewers show you the code created by the visual editor.
